# Sales-Profit-Performance-Dashboard-Tableau
**Overview**
The dashboard provides insights into sales, profit, profit margin, and regional performance to support data-driven decision-making.
**Key Insights**
Technology is the top-performing category
Office Supplies show stable but lower profit
Furniture has lower profitability
Phones and Chairs are top sub-categories
Certain states have high sales but low profit
Overall profit margin is 12.47%, showing scope for improvement.
**Tools & Technologies**
-  Power BI Desktop
-  DAX (for measures and KPIs)
-  Sample Superstore Dataset
